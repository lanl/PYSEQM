seqm.XLBOMD module#

Extended Lagrangian BOMD

XL_BOMD

D: Density Matrix from SP2 algorithm, i.e. given F => D P: Dynamic Density Matrix (field tensor) #t=0 #use SCF loop to construct D, and P(0) = D

P = P(t) F(P) = Hcore + G(P) (Directly construct) D <= F (diagonization or just use SP2)

#electronic energy E(D,P) = (2*tr(Hcore*D) + tr((2D-P)*G(P)))/2.0 here Tr(D) = 2*Nocc #the formula without XL is E(D) = 0.5 * tr(D(Hcore+F)), (Tr(D)==2*Nocc) #agree if P==D

# in A.M.N. Niklasson code, SP2.m return D, tr(D)=Nocc # and E(D,P) = 2*tr(Hcore*D) + tr((2D-P)*G(P))

#in the seqm_functions/scf_loop.py, P means Density matrix
